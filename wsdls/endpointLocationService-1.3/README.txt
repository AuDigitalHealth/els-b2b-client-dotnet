Endpoint Location Service v1.3
================================

These WSDL and XML Schema files are for the v1.2.1 release of the
Endpoint Location Service.

This Endpoint Location Service is described by the Standards Australia
"TR 5823-2010 -- Endpoint Location Service" and NEHTA's "Endpoint
Location Service: Implementation Guide v1.2".

XML Schema files that are referenced by them are also included.
